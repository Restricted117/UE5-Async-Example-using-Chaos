# UE5-Async-Example-using-Chaos
Simple example demonstrating how to use the built in chaos engine in UE5 for async calculations. This example is an actor and must be placed in a level. It counts the number of game ticks, physics ticks, and displays them as a debug message.

Example based off of [this](https://levelup.gitconnected.com/taming-chaos-stable-vehicle-suspensions-with-async-physics-in-ue5-566369c7b097).
